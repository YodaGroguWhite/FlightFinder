A small experimental project for searching cheap flights from Moscow to destinations worldwide using public and affiliate flight APIs.

## What this project is

- Personal, non-commercial prototype
- Used to test flight search APIs (e.g. Travelpayouts / Aviasales)
- Focused on simple, fast, one-way and round-trip searches

## Features

- Origin pre-set to **MOW** (Moscow)
- Destination field for any IATA code or city
- Date-based search for cheapest available options (work in progress)

## Tech stack

- HTML, CSS, JavaScript (no framework)
- Flight search via external APIs (Travelpayouts / Aviasales planned)

## Status

Work in progress.  
Built for learning, experimentation, and API integration tests.
